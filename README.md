# HWE-analysis

This repository contains the code and data for the Hardy-Weinberg Equilibrium (HWE) analysis of gnomAD v4 exome data, including a VCF-to-CSV processing pipeline and all figure generation notebooks.

## Repository structure

```
HWE-analysis/
├── pipeline/
│   └── VCF_to_CSV_Demo_Pipeline.ipynb   # Demo of the VCF processing pipeline
├── figures/
│   ├── Figures_1_S1.ipynb               # Figures 1 and S1
│   ├── Figures_2_S3_S4_TableS1.ipynb    # Figures 2, S3, S4 and Table S1
│   ├── Figures_2l_TableS2.ipynb         # Figure 2l and Table S2
│   └── Figures_S2.ipynb                 # Figure S2
├── scripts/                             # Python modules used by the notebooks
│   ├── HWE.py
│   ├── PowerCalc.py
│   ├── ParsingFiltering_ExomeV4.py
│   └── CorrectingHWE_ExomeV4.py
├── data/
│   ├── pipeline/                        # Reference files for the pipeline notebook
│   │   ├── gnomad_genomes_v3_grch38_AGAIN_highrisk.txt
│   │   ├── gnomad_genomes_v3_grch38_BPHunter_highrisk.txt
│   │   ├── UM_37_Merged_Intervals.csv
│   │   └── MANE_list.txt
│   ├── power/                           # Pre-computed power calculation results
│   │   └── power_*.tsv
│   └── figures/                         # Large gnomAD data files — download from Dropbox (see below)
│       ├── afr_CE_AF0.001_annotated.csv
│       ├── ...
│       └── v4_df_GT_BH_allpop.tsv
└── output/                              # Generated by notebooks (not tracked)
```

## Getting started

### 1. Clone the repository

```bash
git clone https://github.com/khorenponsin-1/HWE-analysis.git
cd HWE-analysis
```

### 2. Download large data files from Dropbox

The gnomAD data files are too large to host on GitHub. Download them from Dropbox:

**Link**: https://www.dropbox.com/scl/fo/gynbp60rk3xxxqn7c1byu/AHu43-aKrGm736z72TXUF1c?rlkey=6xhy66m8k2re42zd7gd8tpu7t&st=e4ngtxet&dl=0

Place the files as follows:

| File | Destination |
|------|-------------|
| `chr21_sample_100k` | `data/pipeline/` |
| `afr_CE_AF0.001_annotated.csv` | `data/figures/` |
| `amr_CE_AF0.001_annotated.csv` | `data/figures/` |
| `asj_CE_AF0.001_annotated.csv` | `data/figures/` |
| `eas_CE_AF0.001_annotated.csv` | `data/figures/` |
| `fin_CE_AF0.001_annotated.csv` | `data/figures/` |
| `nfe_CE_AF0.001_annotated.csv` | `data/figures/` |
| `sas_CE_AF0.001_annotated.csv` | `data/figures/` |
| `v4_df_GT_allpop.tsv` | `data/figures/` |
| `v4_df_GT_BH_allpop.tsv` | `data/figures/` |

### 3. Run the notebooks

Open any notebook from `figures/` or `pipeline/` and run. All output files will be written to `output/`.

## Notes on HGMD

The HGMD (Human Gene Mutation Database) is proprietary and not included — all HGMD annotation columns are set to `0` in the provided data files. To use HGMD annotations in your own analysis, obtain a license at https://www.hgmd.cf.ac.uk/.

## Full pipeline

`chr21_sample_100k` contains the first 100,000 variants of chromosome 21 from the gnomAD v4 exome for demonstration. For production use, download the full gnomAD v4 exome VCF files from https://gnomad.broadinstitute.org/downloads#v4.
