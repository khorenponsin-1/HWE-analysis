# HWE-analysis

This repository contains the code and data for the Hardy-Weinberg Equilibrium (HWE) analysis of gnomAD v4 exome data, including a VCF-to-CSV processing pipeline and all figure generation notebooks.

## Figure S7 — SLiM population divergence simulation

Figure S7 uses forward-time SLiM simulations to assess the impact of population divergence on HWE p-value distributions. The full workflow is:

1. **Run the SLiM simulation** (`scripts/Div_ReScal_5M_Migr.slim`) to generate the raw VCF files. This simulates two populations diverging over 250–2000 generations with migration and is computationally intensive. We have already run it and made the resulting VCF files available on Dropbox (see above) — download them if you want to skip this step.

2. **Place the VCF files** in `data/simulations/` (either from Dropbox or generated in step 1).

3. **Run `figures/Figure_S7.ipynb`** top-to-bottom. It reads the VCFs, converts them to CSV, filters variants (AF > 0.1%), computes HWE and CHWE p-values, and generates the final figures.

Pre-computed CSV files are already provided in `data/simulations/`, so the figures section of the notebook can be run independently without downloading the VCFs.

## Notes on HGMD

The HGMD (Human Gene Mutation Database) is proprietary and not included — all HGMD annotation columns are set to `0` in the provided data files. To use HGMD annotations in your own analysis, obtain a license at https://www.hgmd.cf.ac.uk/.

## Full pipeline

`chr21_sample_100k` contains the first 100,000 variants of chromosome 21 from the gnomAD v4 exome for demonstration. For production use, download the full gnomAD v4 exome VCF files from https://gnomad.broadinstitute.org/downloads#v4.
